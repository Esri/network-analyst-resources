Network Analyst in ArcGIS Pro

ArcGIS Pro is a powerful single desktop GIS application. You can use Network Analyst in ArcGIS Pro in the UI and via python using the arcpy.na and arcpy.nax modules. You can also use ArcGIS Pro to publish your own network analysis services.
