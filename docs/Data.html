Data for Network Analyst

Underlying all network analysis is a network dataset, a routable street network with proper attributes and structure to ensure high quality results. You can purchase a network dataset or create your own using ArcGIS Pro.

StreetMap Premium

ArcGIS StreetMap Premium is Esri's data product optimized for routing, driving directions, and network analysis. You can purchase data for specific countries or regions, and you can request the editable StreetMap Premium Custom Roads version of the product if you want to combine the authoritative street data with your own private roads or other custom data.

Tips and tricks

  • You can use the Create Mobile Map Package tool to convert a file geodatabase network dataset into a non-editable mobile geodatabase network dataset suitable for use in ArcGIS Navigator or in ArcGIS Pro. For some applications, a network dataset in this format has substantially better performance than a file geodatabase network dataset.
  • You can use the Package Map tool to clip an editable file geodatabase network dataset to a specified extent. Add the network to a map in ArcGIS Pro, zoom to the desired extent, and run the Package Map geoprocessing tool, setting the extent to the map's current extent. Then, extract the resulting package file. The geodatabase in the extracted package contains the clipped network dataset. Clipping in this way ensures that all associated feature classes and tables are clipped and maintain correct references and relationships to the network dataset.
  • Download the Find Network Subgraphs tool to identify disconnected network components
