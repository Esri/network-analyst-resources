Navigator for ArcGIS

ArcGIS Navigator is a professional mobile navigation app for Android and iOS that uses GIS software to route your field workforce. ArcGIS Navigator uses the Network Analyst functionality from ArcGIS Runtime for route creation, which supports using travel modes and setting the sequencing of stops. Navigation can work offline using local mobile map packages, which you can either download or create using your own data in ArcGIS Pro.
