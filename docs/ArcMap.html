Network Analyst in ArcMap

ArcMap is a desktop GIS application. You can use the Network Analyst Extension in ArcMap in the UI and via python using the arcpy.na module. ArcMap's functionality is being replaced and superseded by ArcGIS Pro. You are encouraged to migrate to ArcGIS Pro.
